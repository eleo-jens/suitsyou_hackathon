Une API est une interface qui permet à une application comme la notre de récupérer les données d'une autre application. Vous allez utiliser l'API de la NASA pour afficher de magnifiques photos de l'espace.

Il suffit de charger l'url suivant. Mais avant, vous devez faire deux modifications:

  • Affichez 3 images
  • Indiquez le paramètre thumbs à true

https://api.nasa.gov/planetary/apod?api_key=DEMO_KEY&count= &thumbs=

Que faites vous ?

Pas de 3, ni de true: Pas de photos ! :)
